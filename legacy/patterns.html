BLUE MARLIN OS - Migration Component Patterns

This page visually showcases the core UI components and patterns required for each main page type (Landing, Collection, Entry) to guide the Tailwind CSS / 11ty migration. It serves as a visual checklist derived from the project structure and the visual component inventory.
For detailed visual examples and styles, see the UI Style Guide.

Global Patterns / Core Systems

Base Styles & Typography body h1-h6 p a code Foundation styles applied globally. Check Style Guide for details. Style Guide: Typography
Dual Theme System CSS Variables (:root), html.phase-origin toggling Core mechanism for light/dark modes. Variables need mapping to Tailwind theme config. Style Guide: Colors & Themes
Theme Toggle Control .theme-toggle .theme-toggle-knob Interactive element, likely needs JS interaction preserved/re-implemented. Style Guide: Colors & Themes
Underwater Environment Effects .underwater-environment .light-shaft .particle .water-ripple Background effects. Complex; requires careful migration (Tailwind plugin or custom CSS). JS interaction needed. Style Guide: Visual Effects
